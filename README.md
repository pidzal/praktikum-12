### deskripsi proyek
proyek ini mendemontrasikan imlementasi SOLID (SRP, OCP, DIP) pada sistem checkout dengan menggunakan abstraction(interface) dan dependensi injection.

### struktur file
- 'refactor_solid.py' : kode inti yang sudah direfector dan ditambah logging.
- 'README.md' dokumen ini.

### cara menjalankan 
1. pastikan python 3.x terinstal.
2. jalankan file utama dari terminal: 
    ''' bash 
    python refactor_solid.py
    '''
